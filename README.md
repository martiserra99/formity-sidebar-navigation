# Formity — Sidebar Navigation

Starting point for the [Sidebar navigation](https://formity.app/docs/guides/sidebar-navigation) guide on the Formity docs.

## Overview

This repository contains a multi-step form built with Formity and React Hook Form that already includes a sidebar. The sidebar shows the list of steps but does not yet track progress or allow jumping to previously completed steps. The guide walks you through adding that functionality.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-sidebar-navigation
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
